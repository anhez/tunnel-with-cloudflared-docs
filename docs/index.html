# Tunnel với Cloudflared ## Cài đặt To download and install cloudflared: ```bash brew install cloudflared ``` ## Cấu hình Em/Anh sẽ gửi cho mọi người file cấu hình cho từng app, sẽ có 2 file cấu hình: ```config-your-app-name.yml``` và ```xxxx-xxx-xxx-xxxx.json``` Trong file ```config-your-app-name.yml``` thì mọi người cần sửa lại **port** và **đường dẫn chính xác** tới file .json trên thiết bị của mình ## Chạy ``` cloudflared tunnel --config /Users/anhez/.cloudflared/config-your-app-name.yml run your-app-name ``` *Note: hãy đảm bảo đường dẫn config khi chạy lệnh là chính xác * URL của tunnel sẽ có dạng: ```your-app-name.takinato.dev```
